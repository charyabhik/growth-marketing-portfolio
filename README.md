## Tools I Installed

**Cursor IDE**
New to me. Downloaded from cursor.com, standard Windows
install. Took about 1-2 minutes to get comfortable with
the interface — coming from Antigravity (which I used for
building my portfolio site), the layout felt different
but clicked pretty fast.

**Claude Code Extension**
Installed via the Extensions panel in Cursor. I don't
have a Claude Code subscription, so I connected it to
Ollama running locally on my machine as a workaround.
Got it working through the terminal.

**Codex Extension**
Searched and installed directly from Cursor's Extensions
panel. Setup was straightforward.

**Git + GitHub**
Already had a GitHub account — previously connected it
with Lovable for another project. Created a new public
repository for this task.

## Issues I Ran Into and How I Solved Them

**Claude Code subscription**
I don't have a paid Claude Code subscription. Instead of
stopping there, I looked for a workaround — found that
Ollama can run as a local backend, set it up through the
terminal, and got the extension functional.

**Opening the repository in Cursor**
Cloning via terminal was new to me. Looked it up, ran
`git clone` with the repo URL, then opened the folder
directly in Cursor. Worked cleanly after that.

**Creating the README**
Never written one before. Looked up the markdown format
briefly, understood the structure, and wrote it based on
what I actually did during the setup.
